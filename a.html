Tag_A

정의

a태그 (Anchor tag) 는 HTML 에서 굉장히 중요하고 많이사용되는 하이퍼텍스트입니다.
즉 사용자의 선택에 따라 특정한 정보와 관련된 부분으로 이동할수 있게 조직화된 문서를 의미합니다.
주로 다른 웹페이지로 이동할때 쓰이거나 아니면 문서안에 특정 위치로 이동할때 사용합니다.

사용법

<a href=https://www.naver.com/>네이버</a> <- 네이버로 이동 <a href=https://www.google.com/>구글</a> <- 구글로 이동
a태그안에는 이런식으로 href라는 속성이 있는데 이안의 값이 우리가 이동할 주소가 됩니다.
